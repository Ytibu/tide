# 🌊 Tide

> *不争先，争的是滔滔不绝。*

Tide 是一个面向 C++ 的服务器框架，让网络服务开发回归平静与掌控感。  
它不试图成为最快、最“现代”的框架，而是像大海一样：默认就可靠，不需要时时炫耀存在感。

## 📖 为什么叫 Tide？

> *潮汐从不急躁，却从未缺席。*

> 海水连接着每块陆地，却从不需要确认报文。

去掉所有“聪明”的设计，只保留被潮水冲刷过千百遍后依然坚硬的石头，取名为 **Tide**。
